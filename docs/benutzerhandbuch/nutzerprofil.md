# Nutzerprofil {#nutzerprofil}

Das Nutzerprofil enthält persönliche Einstellungen, die pro Nutzer gespeichert werden. Es wird über einen Klick auf den eigenen **Benutzernamen** oben rechts im Desktop geöffnet.

## Nutzerprofil {#tab-nutzerprofil}

Im Tab **Nutzerprofil** können folgende Einstellungen vorgenommen werden:

- **Profilbild (Avatar)**: Über das Auswahlfeld kann ein Profilbild gewählt werden, das in der Kopfzeile des Desktops und in Sofortnachrichten angezeigt wird.
- **Primäre Gruppe**: Zeigt die primäre Gruppe des Nutzers an (nur Anzeige, Änderung über die [Nutzerverwaltung](nutzerverwaltung.md#kuerzel-gruppen)).
- **Kürzel**: Zeigt das Kürzel des Nutzers an (nur Anzeige, Änderung über die [Nutzerverwaltung](nutzerverwaltung.md#kuerzel-gruppen)).
- **Passwort ändern**: Über die Schaltfläche **ändern** kann das eigene Passwort geändert werden. Nach einer Passwortänderung muss die Anwendung neu gestartet werden.

## Benachrichtigungen {#benachrichtigungen}

Im Tab **Benachrichtigungen** wird gesteuert, bei welchen Ereignissen eine E-Mail-Benachrichtigung versendet wird. Voraussetzung ist eine in der [Nutzerverwaltung](nutzerverwaltung.md#stammdaten) hinterlegte E-Mail-Adresse. Der aktuelle Zustellstatus wird am oberen Rand des Tabs angezeigt.

**Benachrichtigung senden wenn:**

| Ereignis | Beschreibung |
|----------|-------------|
| **Erwähnung in Sofortnachricht** | Benachrichtigung, wenn der Nutzer in einer Sofortnachricht erwähnt wurde |
| **Sofortnachricht erledigt** | Benachrichtigung, wenn eine vom Nutzer gesendete Sofortnachricht als erledigt markiert wurde |
| **Kalendereintrag erstellt/geändert** | Benachrichtigung, wenn ein Kalendereintrag für den Nutzer erstellt oder geändert wurde |
| **Termin steht bevor** | Benachrichtigung, wenn ein Termin des Nutzers in Kürze beginnt |
| **Rechnung fällig** | Benachrichtigung, wenn eine Rechnung in einer Akte des Nutzers fällig ist |

## Sicherheit {#sicherheit}

Im Tab **Sicherheit** können sicherheitsrelevante Einstellungen vorgenommen werden:

- **Warnung bei unbekannten Absendern**: Ist diese Option aktiviert, wird vor dem Öffnen von E-Mails unbekannter Absender eine Warnung angezeigt. Dies schützt vor dem versehentlichen Öffnen potenziell schädlicher Inhalte.

## Neue Akten {#neue-akten}

Im Tab **Neue Akten** lassen sich Standardwerte festlegen, die beim Anlegen neuer Akten automatisch übernommen werden:

- **Standard-Eigentümergruppe**: Legt fest, welche Gruppe automatisch als Eigentümergruppe neuer Akten eingetragen wird. Zur Auswahl stehen die Gruppen, in denen der angemeldete Nutzer Mitglied ist.
- **Berechtigte Gruppen**: Über eine Tabelle mit Checkboxen können die Gruppen ausgewählt werden, die automatisch als berechtigte Gruppen neuer Akten eingetragen werden.

Weitere Informationen zur Verwendung dieser Einstellungen finden sich unter [Standardberechtigungen für neue Akten](aktenverwaltung/allgemeine-daten.md#standardberechtigungen).

## Formatierungen {#formatierungen}

Im Tab **Formatierungen** wird die Standardschriftart für formatierte E-Mails und Notizen festgelegt:

- **Schriftart**: Auswahl der Standardschriftart (z.B. Arial, Times New Roman)
- **Schriftgröße**: Auswahl der Standardschriftgröße

Diese Einstellungen werden für den HTML-Editor in E-Mails und Aktennotizen verwendet.

## KI-Werkzeuge {#ki-werkzeuge}

Im Tab **KI-Werkzeuge** können dauerhaft erteilte Genehmigungen für KI-Agenten-Werkzeuge eingesehen und widerrufen werden. Eine ausführliche Beschreibung dieses Tabs findet sich im Kapitel [Genehmigungen verwalten](ki-assistent/agenten.md#genehmigungen-verwalten).
