# Handaktenbogen {#handaktenbogen}

Der Handaktenbogen ist ein druckbares Deckblatt für physische Handakten. Er fasst die wichtigsten Aktendaten auf einer Seite zusammen und eignet sich zum Abheften in Papierakten oder zur schnellen Übersicht.

## Handaktenbogen erstellen

1. Öffnen Sie die gewünschte Akte
2. Klicken Sie auf das Drucker-Symbol in der Werkzeugleiste (Tooltip: "Deckblatt für Handakte drucken")
3. Ein Vorschaufenster mit dem generierten Handaktenbogen öffnet sich
4. Über die Schaltflächen in der Vorschau können Sie den Bogen drucken oder als PDF speichern

## Enthaltene Informationen

Der Handaktenbogen enthält folgende Aktendaten:

| Bereich | Enthaltene Daten |
|---------|-----------------|
| **Grunddaten** | Aktenzeichen, Kurzrubrum, Grund |
| **Zuständigkeit** | Anwalt, Sachbearbeiter |
| **Kostenvorblatt** | Tabelle |
| **Beteiligte** | Namen und Adressen aller Beteiligten |
| **Termine** | Termine, Wiedervorlagen und Fristen |
| **Notizen** | Freitext-Notizen zur Akte |
