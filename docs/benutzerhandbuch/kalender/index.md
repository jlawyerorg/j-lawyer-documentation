# Kalender

Der Kalenderbereich in j-lawyer.org ermöglicht die Verwaltung von Terminen, Wiedervorlagen und Fristen.

## Übersicht der Funktionen

| Bereich | Beschreibung |
|---------|--------------|
| [Aktenbezogener Kalender](aktenbezogen.md) | Termine, Wiedervorlagen und Fristen innerhalb von Akten |
| [Kalendersuche](suche.md) | Aktenübergreifende Suche nach Kalendereinträgen |
| [Fehlende Einträge](fehlende-eintraege.md) | Aktive Akten ohne offene Kalendereinträge identifizieren |
