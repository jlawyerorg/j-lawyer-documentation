# Mobile App

Die j-lawyer Mobile App ermöglicht den mobilen Zugriff auf Ihre Kanzleidaten von unterwegs. Mit der App können Sie Akten einsehen, Dokumente verwalten, Kalendereinträge prüfen und Sofortnachrichten mit Kollegen austauschen.

## Funktionsübersicht

| Bereich | Beschreibung |
|---------|--------------|
| [Installation](installation.md) | App aus dem App Store herunterladen |
| [App koppeln](app-koppeln.md) | Verbindung zum Server herstellen |
| [Dashboard](dashboard.md) | Termine, Fristen und Wiedervorlagen im Überblick |
| [Akten](akten.md) | Aktenübersicht und Detailansicht mit Dokumenten |
| [Messaging](messaging.md) | Sofortnachrichten und Erwähnungen |
| [Einstellungen](einstellungen.md) | App-Konfiguration anpassen |

## Voraussetzungen

- **Android**: Version 14.0 oder höher
- **iOS**: Version 12.0 oder höher
- **Serververbindung**: Die App benötigt eine Verbindung zum j-lawyer Server
    - Bei **j-lawyer.CLOUD** ist dies automatisch gegeben
    - Bei **Self-Hosting** muss der Server erreichbar sein (z.B. über VPN oder Portweiterleitung)

## Offline-Nutzung

Ausgewählte Akten können für die Offline-Nutzung synchronisiert werden. So haben Sie auch ohne Internetverbindung Zugriff auf wichtige Dokumente. Die Synchronisation kann im Desktop-Client oder in der App selbst aktiviert werden - siehe [Akten](akten.md#offline-synchronisation).
