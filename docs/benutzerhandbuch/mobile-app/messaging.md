# Messaging

Die Messaging-Funktion ermöglicht den Austausch von Sofortnachrichten mit Kollegen, auch mit Bezug zu Akten und Dokumenten.

## Nachrichtenübersicht

Die Übersicht zeigt alle Nachrichten des konfigurierten Zeitraums (1 bis 30 Tage, siehe [Einstellungen](einstellungen.md)).

**Funktionen:**

- **Suche**: Durchsuchen Sie Nachrichten nach Inhalt, Aktenbezug oder Erwähnungen
- **Sortierung**: Neueste Nachrichten werden zuerst angezeigt
- **Pull-to-Refresh**: Ziehen Sie nach unten, um neue Nachrichten zu laden

## Nachrichtenkarte

Jede Nachricht wird als Karte mit folgenden Informationen angezeigt:

| Element | Beschreibung |
|---------|--------------|
| **Absender** | Name des Absenders |
| **Datum/Uhrzeit** | Zeitpunkt des Versands |
| **Inhalt** | Nachrichtentext |
| **Aktenbezug** | Verknüpfte Akte (falls vorhanden) |
| **Dokumentbezug** | Verknüpftes Dokument (falls vorhanden) |
| **Erwähnungen** | @-Erwähnungen von Kollegen |

## Erwähnungen (Mentions)

Mit **@-Erwähnungen** können Sie Kollegen direkt ansprechen. Erwähnungen werden farblich hervorgehoben:

| Farbe | Status |
|-------|--------|
| **Rot** | Offen - Sie wurden erwähnt und haben noch nicht reagiert |
| **Grün** | Erledigt - Erwähnung als bearbeitet markiert |

### Erwähnung als erledigt markieren

1. Tippen Sie auf die Nachricht, um die Aktionen anzuzeigen
2. Wählen Sie **"Erwähnung als erledigt markieren"**

## Aktionen

Tippen Sie auf eine Nachricht, um folgende Aktionen auszuführen:

| Aktion | Beschreibung |
|--------|--------------|
| **Zur Akte navigieren** | Öffnet die verknüpfte Akte (falls vorhanden) |
| **Dokument öffnen** | Öffnet das verknüpfte Dokument (falls vorhanden) |
| **Erwähnung erledigen** | Markiert Ihre Erwähnung als bearbeitet |
| **Antworten** | Erstellt eine Antwort auf diese Nachricht |

## Neue Nachricht erstellen

Tippen Sie auf den schwebenden Aktionsbutton, um eine neue Nachricht zu verfassen:

1. **Akte auswählen** (optional): Suchen Sie nach einer Akte, um die Nachricht zu verknüpfen
2. **Empfänger erwähnen**: Tippen Sie auf Benutzernamen, um @-Erwähnungen hinzuzufügen
3. **Nachricht verfassen**: Geben Sie den Nachrichtentext ein
4. **Senden**: Tippen Sie auf Senden, um die Nachricht abzuschicken

### Auf Nachricht antworten

Bei Antworten wird die ursprüngliche Nachricht automatisch zitiert. Der Aktenbezug wird übernommen.

## Offline-Verhalten

Im Offline-Modus können Sie vorhandene Nachrichten lesen, aber keine neuen Nachrichten versenden oder empfangen.
