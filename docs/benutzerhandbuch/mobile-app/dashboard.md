# Dashboard

Das Dashboard zeigt alle anstehenden Kalendereinträge auf einen Blick.

## Übersicht

Das Dashboard gliedert sich in drei Kategorien:

| Kategorie | Symbol | Beschreibung |
|-----------|--------|--------------|
| **Termine** | Kalender | Bevorstehende und vergangene Termine mit Uhrzeit |
| **Fristen** | Ausrufezeichen | Anstehende Fristen (nur Datum, keine Uhrzeit) |
| **Wiedervorlagen** | Glocke | Fällige Wiedervorlagen |

Jede Kategorie zeigt die Anzahl der Einträge an. Tippen Sie auf eine Kategorie, um die zugehörigen Einträge zu sehen.

## Zeitraum konfigurieren

Der angezeigte Zeitraum kann in den [Einstellungen](einstellungen.md) angepasst werden:

- **Vergangene Tage**: 0 bis 30 Tage in der Vergangenheit
- **Zukünftige Tage**: 0 bis 30 Tage in der Zukunft

## Aktualisieren

Ziehen Sie die Liste nach unten (**Pull-to-Refresh**), um die Daten vom Server neu zu laden.

## Schnellaktionen

Über den schwebenden Aktionsbutton (SpeedDial) können Sie direkt neue Kalendereinträge erstellen:

- **Wiedervorlage erstellen**: Neue Wiedervorlage anlegen
- **Frist erstellen**: Neue Frist anlegen
- **Termin erstellen**: Neuen Termin anlegen

## Einträge verschieben

Bei Wiedervorlagen und Terminen können Sie Einträge durch längeres gedrückt halten schnell verschieben:

- **1 Tag** später
- **2 Tage** später
- **3 Tage** später
- **7 Tage** später

!!! note "Hinweis"
    Das Verschieben ist nur bei Wiedervorlagen und Terminen möglich, nicht bei Fristen.

## Offline-Modus

Wenn keine Verbindung zum Server besteht, wird ein entsprechender Hinweis angezeigt. Sie können weiterhin die zuletzt synchronisierten Daten einsehen, aber keine Änderungen vornehmen.
