# Einstellungen

In den Einstellungen können Sie die App an Ihre Bedürfnisse anpassen.

## Darstellung

| Einstellung | Beschreibung |
|-------------|--------------|
| **Animationen** | Aktiviert oder deaktiviert Animationen in der App. Deaktivieren kann die Performance auf älteren Geräten verbessern. |

## Synchronisation

Diese Einstellungen steuern, welche Daten auf Ihr Gerät synchronisiert werden.

| Einstellung | Optionen | Beschreibung |
|-------------|----------|--------------|
| **Max. Dateigröße** | 5 / 10 / 20 / 50 / 200 / 1000 MB | Dokumente, die größer als dieser Wert sind, werden nicht heruntergeladen. Sie bleiben in der Liste sichtbar, können aber nur online geöffnet werden. |
| **Dateitypen** | Mehrfachauswahl | Wählen Sie, welche Dateikategorien synchronisiert werden sollen. |

### Verfügbare Dateikategorien

| Kategorie | Dateiformate |
|-----------|--------------|
| **E-Mails** | .msg, .eml |
| **Dokumente** | .doc, .docx, .odt, .pdf, .txt, .rtf, .csv, .xls, .xlsx, .ods, .ppt, .pptx, .odp, .fodp, .fodt und weitere |
| **Bilder** | .jpg, .jpeg, .png, .gif, .bmp, .svg, .tif, .tiff |
| **Notizen** | .html, .htm, .md |
| **Videos** | .mp4, .avi, .mov, .wmv, .mkv, .webm und weitere |
| **Audio** | .mp3, .wav, .m4a, .aac, .ogg, .flac und weitere |
| **Sonstige** | Alle anderen Dateitypen |

!!! tip "Tipp"
    Beschränken Sie die Dateitypen auf das Wesentliche (z.B. nur Dokumente und E-Mails), um Speicherplatz und Datenvolumen zu sparen.

## Nachrichten

| Einstellung | Optionen | Beschreibung |
|-------------|----------|--------------|
| **Zeitraum** | 1 / 3 / 7 / 14 / 30 Tage | Legt fest, wie weit zurück Nachrichten geladen werden. Ein kürzerer Zeitraum beschleunigt das Laden. |

## Dashboard

Diese Einstellungen bestimmen den Zeitraum der im [Dashboard](dashboard.md) angezeigten Kalendereinträge.

| Einstellung | Optionen | Beschreibung |
|-------------|----------|--------------|
| **Vergangene Tage** | 0 / 1 / 3 / 7 / 14 / 30 | Anzahl der Tage in der Vergangenheit, für die Einträge angezeigt werden. |
| **Zukünftige Tage** | 0 / 1 / 3 / 7 / 14 / 30 | Anzahl der Tage in der Zukunft, für die Einträge angezeigt werden. |

## Allgemein

| Einstellung | Beschreibung |
|-------------|--------------|
| **Sprache** | Wählen Sie die Anzeigesprache der App. Verfügbare Sprachen hängen von der App-Version ab. |
| **Über** | Zeigt Informationen zur App-Version und Lizenzen an. |
