# Installation

Die j-lawyer Mobile App ist kostenlos in den offiziellen App Stores verfügbar.

## Download

### Android (Google Play Store)

[![Google Play Store](https://play.google.com/intl/en_us/badges/static/images/badges/de_badge_web_generic.png){ width="200" }](https://play.google.com/store/apps/details?id=org.jlawyer.mobile&hl=de)

[Direkt-Link: j-lawyer im Google Play Store](https://play.google.com/store/apps/details?id=org.jlawyer.mobile&hl=de)

### iOS (Apple App Store)

[![Apple App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg){ width="170" }](https://apps.apple.com/de/app/j-lawyer-cloud/id6477718420)

[Direkt-Link: j-lawyer im Apple App Store](https://apps.apple.com/de/app/j-lawyer-cloud/id6477718420)

## Nach der Installation

Nach der Installation müssen Sie die App mit Ihrem j-lawyer Server verbinden. Folgen Sie dazu der Anleitung unter [App koppeln](app-koppeln.md).
