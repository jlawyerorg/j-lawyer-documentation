# Installation

Die j-lawyer Mobile App gibt es in zwei Ausprägungen:

| App | Zielgruppe | Kosten |
|-----|-----------|--------|
| **j-lawyer.CLOUD** | Nutzer der j-lawyer.CLOUD | kostenfrei |
| **j-lawyer.COMMUNITY** | Selbstbetriebene j-lawyer-Installationen | Einmalzahlung (kein Abo) |

Beide Apps sind in den offiziellen App Stores verfügbar.

## j-lawyer.CLOUD {#jlawyer-cloud}

Für Nutzer der [j-lawyer.CLOUD](https://j-lawyer.cloud) - kostenfrei.
